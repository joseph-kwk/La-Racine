## Keyboard Navigation

For accessibility, you can also:
1. Tab to the language selector
2. Press Enter to open the dropdown
3. Use Arrow keys to navigate options
4. Press Enter to select
5. Press Escape to close without selecting

## Troubleshooting

### Language Not Changing?
- Make sure you clicked on a different language (not the already active one)
- Refresh the page if needed
- Check your browser console for errors

### Some Text Not Translated?
- This could be user-generated content (tree names, member names, etc.)
- Some backend messages might not be translated yet

### Can't Find Language Selector?
- Make sure you're logged in (it only shows when authenticated)
- Look in the top-right corner of the header
- Try refreshing the page
